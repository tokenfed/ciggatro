# 🚬 CIGGATRO — 游戏规则与玩法指南

*一款以中国香烟文化为主题的 Roguelike 卡牌游戏*

---

## 🎯 游戏目标

在有限回合内，通过打出烟盒卡牌组合，**达到目标分数**即可过关。

通关所有章节，击败最终 Boss，成为**中国卷烟博物馆馆长**！

---

## 🎮 基本流程

```
┌──────────────────────────────────────────────────┐
│  1️⃣ 抽牌 → 手牌补满8张                           │
│  2️⃣ 出牌 → 选1-5张打出，计算得分                 │
│  3️⃣ 检查 → 达到目标分？                          │
│     ├── ✅ 是 → 过关，进入商店                    │
│     └── ❌ 否 → 下一回合（回合用完则失败）        │
│  4️⃣ 商店 → 购买新卡、烟灰缸、消耗品              │
│  5️⃣ 下一关 → 目标分更高，难度递增                │
│  6️⃣ Boss → 每章结尾有Boss战，特殊机制            │
└──────────────────────────────────────────────────┘
```

## 🃏 卡牌类型

### 🚬 烟盒 (主要卡牌)

你的牌组核心。每张烟盒有5个属性：

| 属性 | 说明 | 示例 |
|------|------|------|
| **品牌** | 香烟品牌名 | 中华、玉溪、芙蓉王 |
| **产地** | 生产省份 | 云南、上海、湖南 |
| **年代** | 包装设计年代 | 80s、90s、00s、10s、20s |
| **稀有度** | 星级，影响基础分倍率 | ★ 到 ★★★★★ |
| **基础分** | 卡牌的基础得分 | 10-100 |

**稀有度倍率：**
- ★ Common = ×1.0
- ★★ Uncommon = ×1.5
- ★★★ Rare = ×2.0
- ★★★★ Epic = ×3.0
- ★★★★★ Legendary = ×5.0

---

### 🏺 烟灰缸 (被动效果)

装备后**持续生效**的被动加成卡。最多装备5个。

**常见效果：**
- 固定加分：每回合+X分
- 产地加成：某省份烟+X分
- 品牌加成：某品牌×X倍
- 稀有度加成：某星级烟+X分
- 特殊效果：首发×2、赌博效果等

---

### 🔥 打火机 (消耗品·塔罗牌)

**使用一次后消失**，用来强化你的烟盒。

**常见效果：**
- 给烟盒+基础分
- 让烟盒变成更高稀有度
- 复制烟盒
- 改变烟盒属性

**使用方法：** `use 1 on 3` (用第1个消耗品作用于第3张手牌)

---

### 🗺️ 特产卡 (消耗品·星球牌)

**永久升级某个省份的combo倍率**。

每个省份有专属特产：
- 普洱茶饼 → 云南等级+1
- 臭豆腐 → 湖南等级+1
- 茅台酒 → 贵州等级+1
- ...

省份等级越高，该省份的产地combo倍率越高！

---

### 📦 走私品 (消耗品·幽灵牌)

**稀有强力消耗品**，效果非常强大。

**示例：**
- 海关通行证 — 本局免疫扣押
- 洗钱机 — 烟票×2
- 烟草垄断权 — 所有烟永久+1★

---

### 📜 许可证 (永久升级·优惠券)

在商店购买，**永久生效**的升级。

**示例：**
- 烟灰缸架 — +1烟灰缸槽位
- 批发许可证 — 商店烟盒-20%
- 收银机 — 每回合+3烟票

---

## ✨ Combo 系统 (核心玩法!)

打出多张烟盒时，如果它们有**相同属性**，就会触发combo倍率！

### 品牌 Combo (最重要!)

| Combo | 条件 | 倍率 |
|-------|------|------|
| 对子 | 同品牌 2张 | ×1.5 |
| 三条 | 同品牌 3张 | ×2.5 |
| 四条 | 同品牌 4张 | ×4.0 |
| 一条龙 | 同品牌 5张 | ×8.0 |

**例：** 打出 中华+中华 = 对子 ×1.5

---

### 产地 Combo

| Combo | 条件 | 倍率 |
|-------|------|------|
| 同乡 | 同省份 2张 | ×1.3 |
| 老乡见老乡 | 同省份 3张 | ×2.0 |
| 省队 | 同省份 4张 | ×3.0 |
| 全省制霸 | 同省份 5张 | ×5.0 |

**例：** 打出 玉溪(云南)+云烟(云南)+红塔山(云南) = 老乡见老乡 ×2.0

---

### 年代 Combo

| Combo | 条件 | 倍率 |
|-------|------|------|
| 同龄人 | 同年代 2张 | ×1.2 |
| 年代秀 | 同年代 3张 | ×1.8 |
| 时代印记 | 同年代 5张 | ×3.0 |
| 穿越 | 80s + 20s | ×2.5 |

---

### ⚡ Combo 可以叠加！

这是高分的关键！

**例：** 打出 中华(上海/90s) + 中华(上海/00s)
- 对子 (同品牌): ×1.5
- 同乡 (同产地上海): ×1.3
- **总倍率: ×1.5 × 1.3 = ×1.95**

---

## 📐 得分计算公式

```
最终得分 = (Σ 卡牌基础分 × 稀有度倍率) × 品牌Combo × 产地Combo × 年代Combo + 烟灰缸加成
```

**计算步骤：**
1. 每张牌的分数 = 基础分 × 稀有度倍率
2. 加起来得到总基础分
3. 乘以所有触发的Combo倍率
4. 加上烟灰缸的固定加成

## 🎰 商店系统

每关通关后进入商店，可以用**烟票**购买：

| 类型 | 说明 |
|------|------|
| 🚬 烟盒 | 加入你的牌组 |
| 🏺 烟灰缸 | 装备获得被动效果 |
| 📦 消耗品 | 打火机、特产、走私品 |
| 📜 许可证 | 永久升级 |
| 🗑️ 移除 | 从牌组删除弱卡 |

**烟票获取：**
- 过关奖励
- 部分烟灰缸加成
- 走私品效果

## 👔 Boss 战

每章结尾有Boss战，Boss会有**特殊干扰机制**：

| Boss | 机制 |
|------|------|
| 控烟办主任 | 每回合封印1个品牌(无法得分) |
| 假烟贩子 | 部分手牌变假烟(负分) |
| 海关总署 | 限定烟被扣押 |
| 戒烟意志 | 每打1张，下1张-10% |
| 烟草总局 | 综合所有机制 |

**策略：** 针对Boss机制构建牌组！

## 💡 新手建议

1. **构建主题牌组** — 专注1-2个省份或品牌，更容易触发combo
2. **烟灰缸很重要** — 好的烟灰缸能让得分翻倍
3. **精简牌组** — 移除弱卡，提高抽到强卡概率
4. **省着用消耗品** — 留到Boss战或紧急时刻
5. **注意Boss机制** — 提前准备应对策略
