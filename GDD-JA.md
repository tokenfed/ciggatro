## 1. 概要

### 1.1 ハイコンセプト

**CIGGIES**は、中国タバコ文化をテーマにしたローグライクデッキビルダーゲームです。プレイヤーはタバコパックを集め、灰皿（パッシブ効果）を組み合わせ、高得点コンボを作って攻略します。

インスピレーション：
- **Balatro** — コアゲームプレイフレームワーク
- **ciggies.app** — テーマ、アート、データソース
- **中国タバコ文化** — 社交性、コレクション文化、地域性

### 1.2 コアセールスポイント

- 🎨 **リアルなタバコパックアート** — ciggies.appの高解像度画像を直接使用
- 🗺️ **地域文化** — 各省に独自のタバコスタイルとボーナス
- 🎰 **深い戦略性** — ブランド、産地、年代、タバコ種類の4次元コンボ
- 📦 **コレクション駆動** — メタ進行システム、レアパックのアンロック

### 1.3 対象プラットフォーム

- Web（優先、ciggies.appに直接埋め込み可能）
- iOS / Android（PWA）
- Steam（後期）

### 1.4 ターゲットオーディエンス

- ciggies.appコミュニティユーザー
- Balatraプレイヤー
- 中国文化に興味のあるコレクター志向プレイヤー
- 喫煙者・元喫煙者（ノスタルジア）

## 2. コアループ

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│ ドロー → 選択 → 点火(プレイ) → スコア計算 → 次ラウンド │
│                      ↓                              │
│     ラウンド終了 → ショップ → 購入 → 次ステージ      │
│                      ↓                              │
│     ボス戦 → 撃破 → 新コンテンツ解放 → 次チャプター  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 2.1 シングルラウンドフロー

1. **ドローフェイズ**：デッキから8枚のタバコパックを手札に引く
2. **選択フェイズ**：1〜5枚のパックを選んでプレイ準備
3. **点火フェイズ**：選んだパックをプレイし、効果を発動
4. **計算フェイズ**：コンボ + 灰皿ボーナス → スコア
5. **捨て札フェイズ**：手札の一部を捨てる（回数制限あり）

### 2.2 勝利条件

各ステージには目標スコアがあり、制限ラウンド内に達成すればクリア。

### 2.3 敗北条件

- ラウンド数を使い切っても目標未達
- 特定のボスメカニクスによる敗北

## 7. ボスとステージ

### 7.1 チャプター構造

**第一章：入門（3ステージ + ボス）**
- 目標スコア低め、基本を学ぶ
- ボス：禁煙課長

**第二章：中級（4ステージ + ボス）**
- 産地コンボ導入
- ボス：偽タバコ業者

**第三章：上級（4ステージ + ボス）**
- 年代コンボ導入
- ボス：税関総署

**第四章：コレクター（5ステージ + ボス）**
- 全要素統合
- ボス：禁煙の意志

**隠しチャプター：闇市場（解放条件：全収集）**
- 超高難度
- ボス：煙草総局

### 7.2 ボス詳細設計

#### ボス1：禁煙課長

**背景**：メガホンを持った中年の役人、「禁煙」の看板が背景

**メカニクス**：
- 毎ラウンド1つのブランドをランダムに封印（そのブランドは本ラウンドスコアなし）
- HP：2000点

**攻略**：
- 複数ブランドのデッキを準備
- 封印解除できる灰皿を持つ

**セリフ**：
- 「公共の場での喫煙は条例で禁止されています。」
- 「私も仕方ないんです、上からの指示で。」

---

#### ボス2：偽タバコ業者

**背景**：サングラスの男、倉庫に積まれたタバコ箱

**メカニクス**：
- 毎ラウンド1-2枚の手札を「偽タバコ」に変える（スコアがマイナス）
- 偽タバコは見た目が本物と同じ、細部で見分ける必要あり
- HP：3500点

**攻略**：
- 鑑定系灰皿を持つ（偽タバコを識別可能）
- 捨て札を多用、偽タバコを捨てる

**セリフ**：
- 「これは本物ですよ、この防偽マークを見て...」
- 「安かろう悪かろう？いやいや、これはルートの強みです。」

---

#### ボス3：税関総署

**背景**：制服の税関職員、X線装置が背景

**メカニクス**：
- 限定レアリティのタバコは押収され、プレイ不可
- 2ラウンドごとに1枚解放（ただし税金：-20 チケット）
- HP：5000点

**攻略**：
- 限定カードへの依存を減らす
- またはチケットを多く持って税金に対応

**セリフ**：
- 「申告書をお見せください。」
- 「免税枠を超えています、あちらで手続きをお願いします。」

---

#### ボス4：禁煙の意志

**背景**：抽象的な内面世界、煙が形作る苦悶する人影

**メカニクス**：
- 1枚プレイするごとに、次のカードのスコア -10%（累積）
- 3ラウンドごとにデバフリセット
- HP：8000点

**攻略**：
- リセット前はプレイを控える
- または全力バースト

**セリフ**：
- 「本当にもう一本必要ですか？」
- 「あなたの肺のことを考えて。」
- 「もう3箱目ですよ...」

---

#### 隠しボス：煙草総局

**背景**：巨大なオフィスビル、ロゴがうっすら見える

**メカニクス**：
- 全ボスのスキルを統合
- 5ラウンドごとにメカニクス切り替え

**攻略**：
- 究極の試練、完璧な構築が必要
